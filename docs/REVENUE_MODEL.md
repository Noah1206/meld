## Part 2: 전환 퍼널

### 퍼널 구조

```
Awareness (인지)
    ↓ (바이럴 콘텐츠, HN, Reddit)
Landing Page Visit
    ↓ (30% CTR)
Sign Up (Free)
    ↓ (15% 전환율)
Active User (50 요청 소진)
    ↓ (25% 전환율)
Pro Subscriber
    ↓ (5% 전환율)
Team/Enterprise
```

### 전환율 벤치마크 & 목표

| 단계 | 업계 평균 | Meld 목표 | 근거 |
|------|----------|----------|------|
| Visit → Signup | 5-10% | 8% | 데모 영상 효과 |
| Signup → Active | 20-30% | 25% | 온보딩 플로우 |
| Free → Pro | 3-5% | 5% | 50 요청 제한 |
| Pro → Team | 2-3% | 3% | 팀 기능 필요시 |

### 전환 촉진 전술

**Free → Pro**:
1. 50 요청 도달 시 소프트 업그레이드 모달 (강제 아님)
2. Pro 기능 미리보기 (Figma 연동 시도 → "Pro 필요")
3. 48시간 한정 20% 할인 (첫 결제)
4. 연간 결제 추가 20% 할인

**Pro → Team**:
1. 2인 이상 동일 IP 감지 → Team 제안
2. 협업 기능 데모 (공유 세션)
3. 관리자 대시보드 미리보기

**Team → Enterprise**:
1. 50+ 유저 도달 시 Account Executive 연결
2. SSO 요청 시 Enterprise 안내
3. 보안 설문지 요청 시 Enterprise 전환
